# SV-NAMD

This repository contains structures, scripts, and input/output files supporting the calculations presented in the article "Modulation of Photoexcited Carrier Dynamics by Sulfur Vacancy in CdCl₂/Zn₂S₂ van der Waals Heterostructures"

---

## Repository Structure

- `CdCl2/` — CdCl₂ monolayer calculations.
- `Zn2S2/` — Zn₂S₂ monolayer calculations.
- `Pristine/` — Pristine heterostructure calculations.
- `Single-SV-conf1/` to `Single-SV-conf4/` — Single sulfur vacancy configurations.
- `Dual-SV-near/` — Dual sulfur vacancies at nearby sites.
- `Dual-SV-far/` — Dual sulfur vacancies at distant sites.
- `S-poor/` — Formation energy under S-poor condition.
- `S8/` — Sulfur reference for S-rich condition.
- `NAMD-Pristine/` — NAMD files for the pristine heterostructure.
- `NAMD-SV-conf1/` — NAMD files for the SV-conf1 configuration.

**Software used:** VASP, Hefei-NAMD
